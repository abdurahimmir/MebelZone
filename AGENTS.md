# AGENT RULES

Ты реализуешь production-ready веб-сервис проектирования мебели.

## Главные правила

1. Работай строго по ТЗ (docs папка)
2. Не упрощай требования
3. Работай этапами
4. Каждый этап:
   - цель
   - изменения
   - файлы
   - API
   - БД
   - тесты
5. Не переходи дальше без завершения этапа

## Запрещено

- пропускать этапы
- писать псевдокод
- оставлять TODO без причины
- менять логику ТЗ
- делать "как проще"
- объединять backend/frontend/admin в кашу

## Обязательные этапы

1. Подготовка проекта
2. БД и инфраструктура
3. Авторизация
4. Клиенты и проекты
5. Каталоги
6. Frontend ядро
7. Конструктор
8. Расчеты
9. Экспорт
10. Рендер
11. Админка
12. Монетизация
13. Финализация

## Формат ответа

- этап
- цель
- действия
- код
- проверка
- статус
