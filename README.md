# Furniture Web Constructor

Веб-сервис для проектирования корпусной мебели в браузере.

## Возможности
- проектирование мебели (панели, двери, ящики)
- расчет материалов и фурнитуры
- расчет стоимости
- раскрой
- рендер
- экспорт (JSON, PDF, DXF)
- сохранение проектов
- работа с клиентами
- подготовка к SaaS модели

## Стек
Backend:
- NestJS
- PostgreSQL
- Prisma
- Redis

Frontend:
- Vue 3
- TypeScript
- Pinia
- Three.js / Babylon.js

Admin:
- Vue 3
